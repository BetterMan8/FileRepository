# 📦 个人代码与文件归档仓库

> 这是一个私有仓库，主要用于集中存放、备份和同步我的日常代码及各类项目文件。

## 🗂️ 仓库结构说明
为了避免混乱，本仓库按以下规则组织文件：

- **`/projects`** - 存放完整的项目源代码（如 Web应用、脚本工具等）
- **`/docs`** - 存放文档、笔记、图表及学习资料
- **`/configs`** - 存放配置文件（如 `.vimrc`, `settings.json` 等）
- **`/assets`** - 存放图片、音视频、设计稿等静态资源文件
- **`/archive`** - 存放不常修改的历史备份文件
